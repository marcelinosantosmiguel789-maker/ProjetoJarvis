## рҹҺҜ PRINCГҚPIOS RESPEITADOS

### вң… Domain Isolation
```
Domain
  вҶ’ No external dependencies
  вҶ’ Pure business logic
  вҶ’ Entities with constructors
  вҶ’ Repository interfaces only
```

### вң… Application Layer
```
Application
  вҶ’ Depends only on Domain
  вҶ’ DTOs for external communication
  вҶ’ Services implement business use cases
  вҶ’ AutoMapper for transformations
  вҶ’ No entity returns outside layer
```

### вң… Infrastructure Layer
```
Infrastructure
  вҶ’ Implements Domain interfaces
  вҶ’ EF Core DbContext
  вҶ’ Repository implementations
  вҶ’ No business logic here
```

### вң… API Layer
```
API
  вҶ’ Depends on Application only
  вҶ’ Controllers inject IServices
  вҶ’ Request/Response DTOs only
  вҶ’ No domain entities exposed
```
